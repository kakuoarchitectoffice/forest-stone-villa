# Project Instructions

- This is a static one-page presentation site built with HTML, CSS, and JavaScript.
- Keep the image filenames unchanged and manage slide content through the `slides` array in `script.js`.
- Do not add external libraries unless the user confirms the need.
- Run `npm test` after modifying `script.js`.

## Usage

Open `index.html` in a browser, or host this folder with any static web server.
